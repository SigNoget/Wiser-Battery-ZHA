# LK Wiser Batteritryk – ZHA Blueprint

Dette blueprint giver fuld understøttelse af LK Wiser Batteritryk 4‑slutte via ZHA, med mulighed for indstilling af 4 forskellige tidsbaseret lys-indstillinger.

Øverste knapper styrer gardin.
Nederste knapper styrer op til 2 lys / lysgrupper.
Dim % og Transition tid kan vælges; Dim er ikke perfekt.
